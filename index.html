Règles alimentation clés
Perdre du poids sans se priver
▸Règle des 80/20 : 2 repas plaisir/semaine prévus (vendredi soir + dimanche midi). Pas de culpabilité.
▸Protéines à chaque repas : Skyr, œufs, viande, poisson — ça préserve le muscle pendant la perte de poids.
▸Féculents complets : riz/pâtes/pain complet — cale plus longtemps, moins de fringales.
▸Dîner léger : soupe ou salade le soir — le corps stocke moins si l'estomac est léger la nuit.
▸Fruits de saison : voir l'onglet Saisons dans les Repas — moins cher et meilleur goût.
🍕
Gérer la mal bouffe
Sans culpabiliser
▸Fast food : max 1 fois/semaine, de préférence le midi plutôt que le soir.
▸Nutella/céréales : ok en petite quantité, pas tous les jours.
▸Alcool : très calorique, freine la perte de poids — à limiter au maximum.
▸Après un écart : repas léger le lendemain matin, pas de culpabilité, on repart direct.
▸Sodas : tu bois déjà de l'eau ✅ — c'est l'un des meilleurs réflexes.
😴
Améliorer le sommeil
Pour tes insomnies
▸Cohérence cardiaque : 5 sec inspire / 5 sec expire pendant 5 min avant de dormir. Très efficace.
▸Sport le matin : améliore progressivement le sommeil — évite le soir qui retarde l'endormissement.
▸Heure fixe : même heure de coucher tous les jours, même le weekend.
▸Chambre fraîche : 18–19°C favorise l'endormissement.
▸Lumière le matin : ouvre les volets dès le réveil — régule l'horloge interne.
▸Patience : 3–4 semaines de routine régulière avant de voir l'amélioration.
💧
Hydratation
Simple mais essentiel
▸Objectif : 2L minimum par jour, plus les jours de sport.
▸Au réveil : grand verre d'eau avant la séance — les insomnies déshydratent.
▸Eau gazeuse + sirop citron : parfait ✅ — continue comme ça.
▸Pendant la séance : boire toutes les 10–15 min même si pas soif.
⚖️
Bien se peser
Pour un suivi fiable
▸1 fois/semaine : le lundi matin à jeun, après les toilettes, en sous-vêtements.
▸Balance connectée : les % sont indicatifs (±3–5%). Ce qui compte c'est la tendance.
▸Objectif réaliste : –1 kg/mois en moyenne. Les vêtements changent avant la balance.
▸Photos mensuelles : face + profil, même endroit, même lumière — bien plus parlant que la balance.
🏃
Reprendre la course
Progression en 3 étapes
▸0–3 mois : Marche rapide uniquement — 2×30 min/semaine le soir. Zéro impact.
▸3–6 mois : Méthode Couch to 5K — 1 min course / 2 min marche, progressif sur 8 semaines.
▸6 mois+ : Course continue. Objectif : 5 km sans s'arrêter.
⚠️ Pas de course avant 3 mois — à 111 kg l'impact sur les genoux est trop important pour débuter.
